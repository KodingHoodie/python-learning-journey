# Product Sales Tracker

Automation script that processes daily sales data for an electronics retailer.

## What It Does
Reads a plain text file of product IDs generated by an ERP system, 
cross-references a product database, and outputs a structured CSV report 
with sale ID, date, product name, and unit price.

## Output
- 1000 sales records processed automatically
- CSV report with 5 columns: date, sale ID, product ID, product name, price

## What I Learned
- Reading text files and CSV files with Python
- Working with dictionaries to store and retrieve product data
- Using loops and the datetime library
- Writing structured data to CSV output files
