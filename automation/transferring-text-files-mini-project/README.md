# Transferring Text Files Mini Project

Mini project that reads a CSV file of dad jokes, categorizes each joke 
by its rating score using a custom function, and writes the results to 
a new CSV file with an added rating category column.

## Files
- `dad_jokes.csv` — original dataset with joke ID, joke, and rating
- `modified_dad_jokes.csv` — output file with added rating category column
- `transferring_text_files.py` — main script

## What I Learned
- Reading and writing CSV files with Python
- Using functions to process and categorize data
- Appending new columns to existing CSV data
